Release notes

Most up-to-date yet not always accurate version history, as well as source code for each version, may be found on GitHub Owlbear Rodeo Tracker project page.

January 31, 2022

v 1.3.0

Two major new features are token highlighting in combat and ability to manage data sharing directly from the maps screen - you no longer need to manually create the sticky note.

  • Add: each map has a button to enable or disable data sharing
  • Add: during combat active token is highlighted
  • Add: same initiative values also shown on tracker
  • Add: option to enable/disable active token highlighting
  • Fix: correct display with auto-translate feature in most browsers
  • Fix: few typos in Help article

January 20, 2022

v 1.2.1

Minor update with insignificant changes to style and behavior of help & menu.

  • Fix: minor improvements to popup menu & help window

January 16, 2022

v 1.2.0

This version adds a bunch of new features mostly increasing readability and convenience, including new info/settings tab in combat tracker with recoloring options, and a new detailed Help window, available both from addon popup and from combat overlay.

  • Add: new Info/Settings tab of combat tracker overlay
  • Add: new Help popup window with user guide & changelog
  • Add: show status of the sticky note in Info/Settings
  • Add: option to recolor fog and background
  • Add: option to reset selected tool when toolbar is collapsed
  • Add: visible scrollbars for combat & dice trackers
  • Fix: master/client switch is disabled when data sharing is active
  • Fix: emptying service note no longer produces unclosable editor
  • Fix: combat tracker scroll position no longer resets
  • Fix: minor code & popup menu fixes

January 11, 2022

v 1.1.2

Hotfix to address the issue with combat data randomly getting spawned in wrong sticky notes under specific circumstances.

  • Fix: combat data sharing uses only the correct note

January 07, 2022

v 1.1.1

Minor update with small fixes and improvements, including more mobile-friendly extension menu.

  • Fix: adjust combat tracker refresh rate
  • Fix: minor improvements to popup menu

January 04, 2022

v 1.1.0

A number of updates and fixes, most notably the increase in number of tokens that now can be managed in combat tracker. Data sharing format was updated, which means that this version is incompatible with older OBRT versions and will not share combat tracker data with them.

Also finally got published on Opera Addons store. It took Opera moderation about 2 months to verify, and it's 2 versions behind all other addon stores.

  • Add: option to end combat and keep initiative values
  • Add: localization capabilities for the extension menu
  • Add: Russian localization
  • Add: alter combat buttons behavior
  • Fix: data is reset on map change
  • Fix: more tokens can be stored
  • Fix: combat tracker scrolling
  • Fix: minor code issues

October 23, 2021

v 1.0.3

Minor update to keep up with OBR changes. It required a major revamp of the code that processes sticky notes, through which data is shared between players.

  • Fix: update data sharing for OBR v1.10.1b

October 16, 2021

v 1.0.2

Minor update with small fixes and tweaks mostly to improve detection and prevention of dice sharing mechanic abuse.

  • Add: track dice sharing state, output warning if cheating is suspected
  • Add: highlight same initiative values in editor
  • Fix: properly clear dice tray on open
  • Fix: properly block dice tray during roll

October 12, 2021

v 1.0.1

Minor update with bugfixes. Published on major web stores (Chrome, Edge).

  • Fix: improper storage handling for new installs

October 11, 2021

v 1.0.0 (0.95b)

Minor update with bugfixes. Published on Firefox web store (Addons.Mozilla.Org).

  • Fix: disabling dice log no longer breaks the extension

October 10, 2021

v 0.93b

First release to public, distributed via manually-installable ZIP archives through forums.
